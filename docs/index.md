1. 本系列以 web 前端从业技术为导向, 输出一个标准前端技能 map.
2. 开源作品需要社区的净化和参与, 如有表述不清晰或表述错误, 欢迎[issue 勘误](https://github.com/7kms/standard-frontend/issues). 如果对你有帮助, 请不吝 star.
